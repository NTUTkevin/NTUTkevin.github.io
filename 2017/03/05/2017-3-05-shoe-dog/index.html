shoe dog跑出全世界的人 夢想路上的勇氣與初心

發表於 2017-03-05   |   分類於 reading


在當兵期間閱讀完「shoe dog跑出全世界的人 夢想路上的勇氣與初心- Phil Knight」，這本書是我好友阿張推薦我閱讀

阿張blog阿張的投資理財X生活點滴

前言

閱讀這本書我有兩個動機，第一、想了解nike歷史，在小時候認為穿nike鞋好帥，因此好不容易跟爸爸要一雙nike籃球鞋，特別珍惜它；第二、我想了解Nike如何變成全球企業。

在閱讀前，我以為shoe dog是一本商業管理的書，讓我可以更瞭解大企業的管理與營運。但這本書不僅僅是只是商業管理的書

• 第一、如果以年輕人的角度閱讀它，Knight會以他的故事告訴你「傾聽內心的聲音，尋找自己的志業」

• 第二、如果以創業者的角度去閱讀它，Knight會以他的故事告訴你，「他創業時合作夥伴的特質，當Knight遇到困難時，如何找出創業的初心，Nike賣的不應該只是件商品，而是背後的精神、服務」

• 第三、如果以管理者的角度去閱讀它，Knight會以他的故事告訴你，「與員工、合作夥伴之間，他所採取的管理態度」

• 第四、Knight也為我解答，當一個人，在財富、成就方面登峰造極的成功後，他的心境是如何，在財富自由後Knight是不是還有未完成的事情呢？

不知道這樣是否有吸引到你閱讀這本書呢？ 下面是我整理的值得摘錄片段與閱讀心得

Knight對未來徬徨

對未來徬徨無助，應該是跟我一樣24歲左右的人會遇到的問題吧（除非是富二代ＸＤ）！我該讀研究所還是就業，我要出國工作還是在台灣工作，我以後要做什麼工作？那我們來看看Knight，24歲剛畢業於奧勒岡大學（算不錯），又畢業於頂尖的史丹佛大學商學院碩士的心境吧!

『我說不上自己到底是怎樣的人，也不知道未來可能變成怎樣的人。和所有的朋友一樣，我希望功成名就，和他們不同的是，我不知道成功的定義是什麼。財富？也許吧。娶妻？生子？買房？當然如果我運氣好的話。這些都是我被教導應該追尋的目標，而一部分我也卻對這些心生嚮往（純粹於本能）。但我內心深處，我要的不只是這些，我想要更深刻的東西。我意識到人生苦短，短於我們的認知，短如一次晨跑。我希望自己的人生有意義、有目的、有創意，最重要的是…與眾不同』–(p.023)

剛畢業的Knight，擁有高學歷的他，也是一樣對未來徬徨無助，但他知道他希望人生有意義、與眾不同，想在世界上留下足跡。

哈哈，現在快退伍的我，其實對未來滿樂觀的，因為我相信我的人生有意義、我也要與眾不同。

瘋狂點子

假設你現在就是Knight，用有高學歷、對未來徬徨無助，但你想要與眾不同你該怎麼呢？

「擁有瘋狂點子」
就讓別人說我的想法瘋狂吧…
繼續跑下去就對了，永遠別停腳
甚至連想都不要想，直到抵達那兒
千萬不要把過多的注意力放在「那兒」是哪裡
不管碰到什麼，繼續跑下去就對了—（p.021）

當時Knight的瘋狂點子是，他在史丹佛修課時，認真分析的一個報告，“引進日本虎牌鞋（現在叫asics）到美國”，當他上台報告時，並沒有受到同學與教授們的熱烈反應，但在他心中始終覺得或許可行。
（Ps. Knight創辦藍帶體育公司代理虎牌鞋的在美國西岸銷售，後來虎牌鞋的公司鬼塚想另找代理商，Knight因此將公司改Nike，做自己的品牌 Ｗiki:https://zh.wikipedia.org/wiki/%E8%80%90%E5%85%8B ）
Knight想要人生有有意義，與眾不同，想要在世界上留下足跡。他說，若沒出國看看，怎麼能在世界上留下足跡呢？他下定決心去環遊世界，並順路到日本取的虎牌的代理銷售權，於是他就帶著他所有的積蓄、父親資助的一些錢就出發。

瘋狂點子＝志業

何謂『瘋狂點子』其實就是找到自己的『志業』

『瘋狂點子』＝『志業』

在書中最後時，Knight在人生回顧時，他與年輕人分享經驗

『我會要他們按下暫停，長遠而用心思考自己想要什麼運用時間，未來四十年想要跟誰度過。我會告訴25歲左右的男女，不要隨便找個工作或行業，甚至是職業。要找的是志業，就算不知道那是什麼意思，去找就對了。假如跟著志業走，疲憊會比較容易熬過去，絆腳石會成為燃料，高牆則會讓你壓根就感覺不到』—(p.442)

如何尋找志業

如何找到自己的志業呢？在生活中做嘗試各種事情，你嘗試完後會有兩個結果，我喜歡做這個事情 或 我不喜歡做這個事情，如果嘗試後喜歡做這件事情，恭喜你這就是你的『志業』，朝向那兒奮力邁進吧！但如果你嘗試出來，你不喜歡做這件事的話，沒關係，你也發現你不喜歡做這件事，將來你就別選這個當你的志業。
我認爲不一定要創業才能當志業，就業也能當志業。
（想要瞭解更多可以看 葉丙成教授的“如何探索有意義的人生？14’40 ”https://www.youtube.com/watch?v=SqNK5mgeH-4）
接著Knight始終相信，他經營的藍帶公司跟後來的創立的Nike是他的志業，一步一步的建立Nike帝國，並打敗adidas。
在書中，Knight很精彩的敘述整個創立藍帶與Nike過程，但我就不加以敘述，請大家自己去看～

運氣

最後，也是最重要的是 Knight說

『『我想要公開坦承運氣的威力。運動員要靠運氣、詩人要靠運氣、生意要靠運氣。努力不懈至關重要，好的團隊不可或缺，頭腦跟決心無比珍貴，但決定結局的或許是運氣，你越努力不懈你的運氣也越強。要對自己有信念，但也要對信念有信念，不是別人所定義的信念，而是你所定義的信念。是信念在你心中定義自己的信念。』—(p.443)

再往志業努力時，也是需要一點運氣推你邁向成功。
其實也不用擔心運氣不會來，我相信吸引力法則，只要你相信你的志業，運氣隨之而來的。

這本書，我之前都是口頭上推薦給朋友的，但我有一個非常好的朋友，對未來非常徬徨，因此我寫這篇文章給他，一起加油！
